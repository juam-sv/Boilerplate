# Projects for Dio.me
Laboratorios e scripts feito para as aulas acompanhadas no dio

## Installation

Use o [vagrant](https://www.vagrantup.com/) para rodar o ambiente.

```bash
sudo pacman -S vagrant
```

## Usage

```python
vagrant up

# provisionar somente o playbook
vagrant provision 

```
